# Ecommerce Sales Analysis Dashboard

A Power BI dashboard analyzing ecommerce sales performance across states, 
categories, payment modes, and time periods.

## 📊 Dashboard Overview
This dashboard tracks key sales metrics and helps identify top-performing 
states, customers, product categories, and payment trends.

## 🔧 Tools Used
- Power BI Desktop
- DAX (basic measures)
- Power Query (data cleaning)

## 📈 Key Metrics (KPIs)
- Sum of Amount
- Sum of Profit
- Sum of Quantity
- Average of Person

## 🧩 Dashboard Features
- **Filters:** Quarter (Qtr 1–4), State
- **Top 4 State** – sales by state (bar chart)
- **Top Customer** – highest value customers (bar chart)
- **Quantity by Payment Mode** – COD, UPI, EMI, Credit/Debit Card (donut chart)
- **Quantity by Category** – Clothing, Electronics, Furniture (donut chart)
- **Profit-Loss by Month** – monthly profit/loss trend (bar chart)
- **Profit by Sub-Category** – Printers, Sarees, Bookcases, etc. (bar chart)

## 💡 Key Insights
- Maharashtra is the top contributing state by sales amount
- COD is the most preferred payment mode (44%)
- Clothing leads as the top-selling category (63% share)
- Profit shows seasonal dips in mid-year months
